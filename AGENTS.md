# AGENTS.md

## 目的
一般的な AI エージェント向けの共通作業方針を定義します。

## 基本方針
- 会話言語: 日本語
- コード内コメント: 日本語
- エラーメッセージ: 英語
- コミットメッセージ: Conventional Commits（説明は日本語）

## 判断記録のルール
重要な設計判断や変更を行った際は、その理由、検討した代替案、前提条件を明記してください。

## 開発手順
1. プロジェクトの構造と `src/` 配下の各ファイルの役割を理解する
2. `pnpm install` で依存関係をセットアップする
3. 変更を実装し、日本語でドキュメント（JSDoc 等）を記述する
4. `pnpm test` および `pnpm lint` で品質を確認する

## セキュリティ / 機密情報
- Discord の Webhook URL やトークンなどの認証情報をコードやコミットに含めない
- ログに機密情報を出力しない

## リポジトリ固有
- `data/` ディレクトリ配下のファイル（`config.json`, `notified.json`）は実行時に生成・使用される
- Docker 環境での動作（`compose.yaml`, `Dockerfile`）も考慮する
